# Bloom Branding Website

A React-based website for Bloom Branding.

## Setup
1. Run `npm install` (already done).
2. Run `npm run dev` to start the development server.
3. Open `http://localhost:5173`.

## Structure
- **Framework**: React + Vite
- **Styling**: Vanilla CSS (Variables in `src/index.css`)
- **Animation**: Framer Motion
- **Icons**: Lucide React

## Font Guide
- Brand: Playfair Display (Alternative for Bigilla)
- Subtitles: Lekton
- Body: Arial Nova/Arial

## Color Palette
- Earl Gray: #e8e6d8
- Electric Blue: #892f1a
- Butter Yellow: #bdaf62
- Dark Chocolate: #624a41
